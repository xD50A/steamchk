# Steam-Account-Checker
Steam Account Checker

## Installing

Requirements: <br/>

<li> pycryptodome </li>

```bash
pip install pycryptodome
```
<li> requests </li>

```bash
pip install requests
```
<li>loguru</li>

```bash
pip install loguru
```
<li>colorama</li>

```bash
pip install colorama
```
<li> python 3.7 </li>
<br/>
Commands:

```bash
git clone https://github.com/menkrep1337/XSSCon
cd steamchk
pip install -r requirements.txt
python3 steamchk.py 
```
